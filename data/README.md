# Datos temporales para pruebas

Coloca aqui el archivo CSV de prueba con este nombre exacto:

`asegurados.csv`

Encabezados esperados:

- `NumeroPoliza`
- `GeneroAsegurado`
- `nombrecompleto`
- `GrupoEconomico`
- `dGrupoAseguradora`
- `Parentesco_Descriptivo`
- `EDAD`

Notas:

- El archivo `data/*.csv` esta ignorado por Git para no subir datos sensibles por accidente.
- Este CSV sera solo una fuente temporal de pruebas. Despues podremos cambiarlo por una API sin rehacer el flujo del bot.
